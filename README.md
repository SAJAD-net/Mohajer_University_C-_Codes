## GCD (Greatest common divisor)
 #### از **GCD (Greatest Common Divisor) یا بزرگترین مقسوم علیه مشترک** برای پیدا کردن بزرگترین مقسوم علیه مشترک دو عدد استفاده می شود.
## Kth_Smallest
#### پیدا کردن k امین عدد کوچک در یک آرایه نامنظم.
